什么是 JNU-EXAM

JNU-EXAM 是一个通过 Github/Gitee/Cloudflare R2 等方式，存储并共享江南大学考试资料的开源共享平台，旨在通过在线存储，系统性地收集、整理并长期维护包括历年试卷、课程笔记、复习提纲在内的多种学习资源，帮助在校学生完成课程学习。
由于资料众多，不能保证每个文件内容正确，甚至有的文件可能无法打开。遇到资料错误请及时联系群主。

怎么用 JNU-EXAM

目前提供如下途径下载资料

  • 推荐：使用 QQ群(757343447) 群文件中两款下载器 JNU-EXAM-Downloader.exe 或 期末无挂.apk 下载

    • 对于电脑，如果使用 Windows 系统，且 CPU 架构为 x64，请使用 JNU-EXAM-Downloader.exe
    • 对于手机，如果使用 安卓 系统，请使用期末无挂.apk

  • 如果不希望使用下载器，可以通过 Github 仓库下载：https://github.com/gubaiovo/JNU-EXAM

  • 如果不希望使用如上两种方式，可以关注 QQ群文件，会不定期将所有资料打包为压缩包并上传到 QQ群文件 中

  • 如果上面三种都不可以，那就找群主要（可能会看不见）

Q: 为什么我下载不了文件 / 打不开Github / 有“出现网络问题“之类的提示

  1. 切换网络，重新尝试

  2. 换源，使用其他源下载

  3. 科学上网

什么是科学上网：引用知乎解释：通过特定技术手段绕过网络审查或地理限制，访问通常无法直接访问的互联网资源。
由于 Github / Cloudflare R2 服务器均位于海外，国内访问速度时好时坏，因此有时可能需要通过科学上网来加速下载。
如果你不理解这是什么意思，那就到群文件找资料，或者找群主要(最好把出现问题的截图也一起发来)。

Q: 什么是 Github / Gitee / Cloudflare R2

GitHub 是一个在线软件源代码托管服务平台，主要用于公开程序或软件的代码。
Gitee 可以认为是国产的 Github。
Cloudflare R2 是 Cloudflare 推出的对象存储服务。
如果你不想理解上面这些名词概念，那只需要把这三个当作功能多一些的网盘就可以

Q: 投稿是怎么处理的

  1. 将资料放到群文件 “投稿” 文件夹

  2. 不定期提取 “投稿” 文件夹中全部内容，进行文件处理(如：word 转 pdf，打水印等等)，并上传至平台

  3. 清空 “投稿” 文件夹
